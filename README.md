# HCCA
Hot Chocolate Coffee Alternative mod, for Stardew Valley

This is just the Visual Studio portion of the mod. To edit the whole mod, you need to download the latest version of the mod from Nexus, at: https://www.nexusmods.com/stardewvalley/mods/16159
The compiled VS portion of the mod should go in the HCCA Main folder of the mod. I recommend deleting the contents of the HCCA Main folder, and moving in the new files after, instead of replacing the files.
